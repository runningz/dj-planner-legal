Política de Privacidad — DJ Planner

Última actualización: 24/02/2026

Responsable: Rubio AppStudio · Contacto: rubio.appstudio@gmail.com

Resumen: DJ Planner trata los datos mínimos necesarios para funcionar (cuenta, eventos y suscripciones PRO). No usamos analítica de terceros para rastreo publicitario. No vendemos tus datos.

1) Responsable del tratamiento

El responsable del tratamiento de los datos personales tratados en la aplicación DJ Planner es Rubio AppStudio.

Email de contacto: rubio.appstudio@gmail.com

2) Datos que recopilamos

Según el uso que hagas de la app, DJ Planner puede tratar:

  • Datos de cuenta (login con Google via Firebase): identificador de usuario, email y nombre/imagen de perfil si Google los proporciona y el usuario los tiene configurados. DJ Planner utiliza un ID técnico asociado a tu cuenta para operar el servicio.
  • Datos que introduces en la app: eventos, fechas, ubicaciones/direcciones, notas, servicios, importes y estado de pagos (por ejemplo, “pagado/parcial/pendiente”).
  • Datos técnicos necesarios para el funcionamiento: versión de la app, sistema operativo, idioma y registros de diagnóstico en caso de errores o incidencias.
  • Datos de compras/suscripción (PRO): estado de suscripción, producto contratado y eventos de compra/restauración. Los datos de pago completos los gestiona la plataforma (Google Play). DJ Planner no almacena números de tarjeta.

3) Datos que NO recopilamos

  • No recopilamos deliberadamente datos sensibles (salud, biometría, ideología, etc.).
  • No usamos la información para publicidad personalizada.
  • No vendemos tus datos personales.

4) Finalidades (para qué usamos tus datos)

  • Prestación del servicio: autenticar tu cuenta, guardar y sincronizar tus eventos y configuración.
  • Funcionalidades PRO: habilitar el acceso a características de pago, compras y restauraciones.
  • Soporte: responder consultas y resolver incidencias.
  • Seguridad y estabilidad: prevenir abuso y mantener la integridad del sistema.

5) Base legal

  • Ejecución del contrato: para prestar las funcionalidades principales de la app.
  • Interés legítimo: para seguridad, prevención de fraude y mejora técnica/estabilidad.
  • Consentimiento: cuando corresponda (por ejemplo, permisos del dispositivo como ubicación para navegación).

6) Proveedores (terceros encargados de tratamiento)

Para operar DJ Planner se utilizan proveedores tecnológicos que pueden tratar datos en nuestro nombre:

  • Google Firebase: autenticación (Firebase Auth con Google) y almacenamiento/sincronización de datos (Cloud Firestore).
  • RevenueCat: gestión del estado de suscripción, compras y restauraciones para funcionalidades PRO.
  • Google Play: procesamiento de pagos y gestión de suscripciones a nivel de plataforma.

Estos proveedores pueden procesar datos desde diferentes países. Cuando aplica, se utilizan mecanismos adecuados para transferencias internacionales de datos.

7) Ubicación y GPS

DJ Planner puede usar la ubicación del dispositivo solo cuando el usuario lo solicita para guiar la navegación hacia la ubicación de un evento (por ejemplo, abrir un mapa o ruta).

  • No se utiliza ubicación en segundo plano.
  • No se realiza seguimiento continuo de ubicación.
  • Puedes conceder o revocar el permiso de ubicación desde los ajustes del dispositivo.

8) Permisos del dispositivo

DJ Planner puede solicitar permisos únicamente para habilitar funcionalidades concretas:

  • Notificaciones: para recordatorios de eventos.
  • Ubicación (opcional): para navegación a un evento cuando el usuario lo solicita.

Siempre puedes gestionar permisos desde los ajustes del sistema.

9) Conservación (retención) de los datos

Conservamos los datos mientras exista tu cuenta o mientras sea necesario para prestar el servicio. También podemos conservar información mínima durante un periodo razonable para:

  • Cumplir obligaciones legales (por ejemplo, relacionadas con compras, si aplica).
  • Resolver disputas y prevenir fraude.
  • Garantizar seguridad del servicio.

10) Seguridad

Aplicamos medidas técnicas y organizativas razonables para proteger los datos (por ejemplo, cifrado en tránsito y controles de acceso). Aun así, ningún sistema puede garantizar seguridad absoluta.

11) Derechos del usuario

Puedes solicitar acceso, rectificación o eliminación de tus datos, así como oponerte o limitar ciertos tratamientos cuando sea aplicable.

Cómo ejercerlos: escribe a rubio.appstudio@gmail.com indicando el email asociado a tu cuenta y tu solicitud (por ejemplo, “eliminar mi cuenta y mis datos”).

12) Eliminación de cuenta y datos

Si la app incluye una opción de eliminación dentro de ajustes, puedes usarla. Si no, puedes solicitarla por email. Eliminaremos o anonimizaremos los datos salvo aquellos necesarios para cumplimiento legal o prevención de fraude.

13) Privacidad de menores

DJ Planner no está dirigida a menores. No recopilamos deliberadamente datos de menores.

14) Cambios en esta política

Podemos actualizar esta política por cambios legales, técnicos o de producto. Publicaremos siempre la versión vigente en esta misma URL indicando la fecha de actualización.

15) Contacto

Para cualquier consulta sobre privacidad: rubio.appstudio@gmail.com

Esta política describe el tratamiento general de DJ Planner. Algunas integraciones (por ejemplo, mapas/navegación) pueden depender de aplicaciones o servicios del dispositivo del usuario, que se rigen por sus propias políticas de privacidad.
